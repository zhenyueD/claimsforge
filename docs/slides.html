Showcase 2 — Policy-Grounded Reasoning

💰 CompensationAgent cites its rules.

10 policies (data/policies.json)

  • P-RET-01 severity ≥7 → full refund (cap ¥500)
  • P-RET-02 severity 4-6 → 50% refund
  • P-RET-04 易碎品 + crack ≥5 → replacement
  • P-RET-05 电子产品进水 → force escalate
  • P-EMO-01 emotion ≥8 → +20% uplift
  • P-LMT-01 no image + ≥¥50 → escalate

Hybrid filter → LLM proposal

Deterministic filter narrows policies by severity / type / emotion. Gemini picks the best fit and writes an empathetic justification:

"我们很抱歉您收到的马克杯有裂纹。考虑到商品损坏情况，我们将为您办理全额退款 ¥24.00..."
